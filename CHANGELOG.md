# Changelog

## [1.0.0] - 2026-01-16

### Added
- Initial release of standalone Grabby Inspector
- Vite adapter with middleware plugin
- Next.js adapter supporting both App Router and Pages Router
- Automated setup scripts for both frameworks
- Client-side HUD with precision element selection
- DOM navigation with keyboard controls
- Element stack cycling with scroll
- Rich data capture (selectors, styles, dimensions, attributes)
- Viewport-aware tooltip positioning
- Spotlight blur effect for focused inspection

### Features
- Framework-agnostic client script
- Plug-and-play installation
- Zero-dependency browser HUD
- AI-assistant integration ready
- TypeScript support
